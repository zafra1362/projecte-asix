Projecte d’Automatització amb Ansible i Docker

Introducció

Aquest projecte té com a objectiu desplegar una infraestructura contenitzada i automatitzada mitjançant Ansible i Docker, afegint-hi mecanismes de recuperació automàtica i desplegament al núvol.

Serveis Implementats

  • Apache: Servei web que allotja el portal informatiu del projecte.
  • PostgreSQL: Base de dades relacional per gestionar dades estructurades.
  • Postfix: Servei SMTP per enviar correus electrònics interns.
  • Prometheus: Sistema de monitoratge per recollir metrics dels serveis.
  • Grafana: Plataforma per visualitzar gràfics i dashboards a partir de dades de Prometheus.
  • cAdvisor: Monitoratge de l'ús de recursos dels contenidors Docker.
  • Alertmanager: Gestió d'alertes generades per Prometheus davant d'incidències.
  • Node Exporter: Recollida de metrics de sistema de la màquina amfitriona.

Objectius del Projecte

  • Automatitzar el desplegament i gestió de serveis amb Ansible.
  • Contenitzar serveis utilitzant Docker.
  • Simular fallades i comprovar la capacitat d'autorecuperació.
  • Monitoritzar serveis i infraestructura amb Prometheus i Grafana.
  • Desplegar i escalar serveis al núvol.

Metodologia

Es fa servir una metodologia àgil (Scrum) per planificar i iterar el desenvolupament. Es controlen les tasques mitjançant Trello, GitHub i es fa seguiment amb Grafana i Prometheus.

Tecnologies Utilitzades

  • Ansible: Automatització i gestió d'infraestructures.
  • Docker: Contenització d'aplicacions i serveis.
  • Prometheus: Monitoratge i alertes.
  • Grafana: Visualització de dades de monitoratge.
  • cAdvisor: Estadístiques d'ús de contenidors.
  • Alertmanager: Gestió i notificació d'alertes.
  • Node Exporter: Metrics del sistema operatiu.
